# Nigerian Education Survey Analysis (Power BI)

This project presents a data-driven analysis of a nationwide education survey conducted among key stakeholders in Nigeria’s education system, including teachers, students, parents/guardians, and school administrators.

The goal of the project is to uncover major challenges, perceptions, and inequalities within the Nigerian education sector using interactive Power BI dashboards.

# Project Objectives

This analysis aims to:

Understand how different stakeholders perceive the quality of education in Nigeria
Evaluate teaching quality, classroom comfort, and access to learning materials
Assess access to digital tools for teaching and learning
Examine school fee affordability among parents
Identify the top challenges facing Nigerian education
Explore differences across school type, location, age group, and gender
# Key Insights

Some major findings from the survey include:

Only 35% of parents consider school fees affordable
→ Nearly 2 out of 3 parents struggle to pay school fees.
The biggest challenges facing Nigerian education:
Funding & affordability
Poor infrastructure & learning environment
Teacher shortage & quality issues
Government & policy failures
Lack of practical learning
Urban schools dominate responses, highlighting possible rural underrepresentation and inequality.
Teachers rated access to digital tools as moderate, with rural schools showing lower access compared to urban and semi-urban areas.
Students in mission schools rated access to learning materials highest, while government schools recorded the lowest ratings.
🛠 Tools Used
Power BI – Data cleaning, modeling, DAX measures, and dashboard creation
Power Query – Data transformation and categorization
Survey Data – Responses from 230+ education stakeholders
# Dashboard Features

The Power BI dashboard includes:

KPI cards for key education ratings
Interactive slicers by respondent type
Visual breakdown by:
Location (Urban / Semi-Urban / Rural)
School Type (Government / Private / Mission)
Age group and gender
Access to digital tools
Top perceived education challenges
# Why This Project Matters

Education is a critical driver of national development.
This project demonstrates how data analytics can be used to highlight systemic issues and support evidence-based decision-making in the education sector.
